# Persona AI

Persona AI is a web application that lets users chat with AI-powered versions of popular Indian tech educators, Hitesh Choudhary and Piyush Garg. The project simulates a real-time conversation experience, mimicking the creators' unique personalities, Hinglish language styles, and mentorship approaches.

## Features
* **Two Personas:** Switch between Hitesh Choudhary (focusing on "Chai aur Code" and practical analogies) and Piyush Garg (focusing on building, shipping, and direct learning).
* **Smart UI:** A beautiful dark-mode interface styled with modern glassmorphism, dynamic glowing effects, and a responsive sidebar.
* **Interactive Chat:** Includes clickable suggestion chips, message history, typing indicators, and support for formatted code blocks.

## How it Works
* **Frontend:** Built with index.html, style.css, and client.js to handle the user interface and chat interactions.
* **Backend:** Powered by a Node.js and Express server that connects to an OpenAI-compatible API.
* **Prompt Engineering:** Tailored system instructions guide the AI model to reply in a mix of Hindi and English, using specific catchphrases like "Hanji" and "Chai piyo code karo" to sound exactly like the educators.

## Project Goal
The application aims to bridge the gap between students and mentors by offering accessible, contextualized guidance. By using native phrasing and relatable teaching analogies, the simulated mentors make complex coding concepts feel friendly and conversational for young developers.
